# Документирование требований для системы подбора персонала

## Описание
В данном проекте мною были задокументированы требования (функциональные и нефункциональные) для системы подбора персонала

## Содержание
- FSD документ с функциональными и нефункциональными требованиями
- Документ с расписанными в текстовом формате Use Cases
- UML диаграммы:
  - Use Case
  - Sequence
  - Activity
